# Telegram-бот «Прайс Мастер» (Агрегатор цен на iPhone)

Бот в реальном времени сравнивает цены на разные модели iPhone между розничными магазинами и Яндекс Маркетом с учетом партнерских промокодов.

### Использованные технологии:
* **Python 3.11**
* **Aiogram 3.x** — асинхронный фреймворк для Telegram-ботов
* **BeautifulSoup4 & Requests** — парсинг розничного каталога
* **SQLite3** — база данных для кэширования цен маркетплейса и защиты от блокировок по API
* **Asyncio** — запуск тяжелых процессов парсинга в фоновых потоках без блокировки бота

### Архитектура проекта:
* `main_bot.py` — точка входа и инициализация бота
* `handlers/iphones.py` — логика интерактивного меню и вывода результатов
* `parsing/parser.py` — алгоритмы сбора данных с умной фильтрацией моделей
* `database/` — локальное хранилище данных

### Как запустить проект:
1. Запусти скрипт обновления базы данных (создаст таблицу и подтянет цены):
   ```bash
   python update_yandex.py
   ```
2. Запусти основного Telegram-бота:
   ```bash
   python main_bot.py
   ```
