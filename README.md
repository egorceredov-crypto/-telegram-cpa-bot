# 📱 Telegram-бот «Прайс Мастер» (Агрегатор цен на iPhone)

Асинхронный Telegram-бот, который сравнивает цены на различные модели iPhone между популярными розничными магазинами и Яндекс Маркетом с учетом актуальных партнерских промокодов.

---

## 🛠 Технологический стек

* **Python 3.11** — основная языковая платформа.
* **Aiogram 3.x** — современный асинхронный фреймворк для взаимодействия с Telegram API.
* **BeautifulSoup4 & Requests** — инструменты для парсинга и сбора данных из каталогов розничных магазинов.
* **SQLite3** — локальная база данных для кэширования цен маркетплейса (защищает от частых запросов и блокировок по API).
* **Asyncio** — управление асинхронными задачами и запуск фонового анализа цен без зависания интерфейса бота.

## 🚀 Инструкция по запуску

### 1. Подготовка окружения
Клонируйте репозиторий и перейдите в папку проекта:
```bash
git clone https://github.com
cd price_master_bot
```

Создайте файл `.env` в корневой директории и укажите ваш токен бота:
```env
BOT_TOKEN=ваш_токен_от_botfather
```

### 2. Инициализация базы данных
Перед первым запуском бота необходимо создать таблицы и загрузить актуальные цены с Яндекс Маркета:
```bash
python update_yandex.py
```

### 3. Запуск Telegram-бота
Для старта бота выполните команду:
```bash
python main_bot.py
```

## 💡 Функционал
* 📊 **Сравнение цен:** Автоматический сопоставительный анализ цен ритейлеров и маркетплейса.
* 🎟 **Учет промокодов:** Бот автоматически вычитает скидки по партнерским промокодам из итоговой стоимости.
* ⚡ **Фоновое обновление:** База данных обновляется в фоновом режиме, не мешая пользователям отправлять запросы.
